# Qiskit-QCNN
## Qiskit Quantum Convolution Neural Network Jupyter Notebook

---

**Description:** Example code and guides for using with the RasQberry Two.

### Setup Instructions

1. Open a new **Terminal** window on the Raspberry Pi.

2. Ensure yo are in the 'RQB2' virtual environment, or activate it using:

   ```bash
   source ~/RasQberry-Two/venv/RQB2/bin/activate
   ```

3. Clone this repository to your Raspberry Pi:

   ```bash
   git clone https://github.com/jzasjam-shu/Qiskit-QCNN.git
   ```

5. Enter the repository directory:

   ```bash
   cd Qiskit-QCNN
   ```

6. Start Jupyter Notebook:

   ```bash
   jupyter-notebook
   ```

7. Open your browser and navigate to:

   ```
   http://localhost:8888
   ```

   You should now see the notebook file contained in this repository (the .ipynb file). Click on one this to open the notebook and run the code examples.
