# Qiskit-QCNN
## Qiskit Quantum Convolution Neural Network Jupyter Notebook

---

**Description:** Example code and guides for using with the [RasQberry Two](https://rasqberry.org/) with [Qiskit](https://qiskit-community.github.io/qiskit-machine-learning/tutorials/index.html) 

### Setup Instructions

### 1. Open a new **Terminal** window on the Raspberry Pi.

   <img width="511" height="607" alt="image" src="https://github.com/user-attachments/assets/94b7f5f6-4216-4f2d-9679-1b416e1d1ed0" />

---

### 2. Ensure you are in the 'RQB2' virtual environment, or activate it using:

   ```bash
   source ~/RasQberry-Two/venv/RQB2/bin/activate
   ```

   <img width="612" height="100" alt="image" src="https://github.com/user-attachments/assets/e3d1dc78-5386-41f6-8c95-8cbe1d157961" />

---

### 3. Clone this repository to your Raspberry Pi:

   ```bash
   git clone https://github.com/jzasjam-shu/Qiskit-QCNN.git
   ```

   <img width="611" height="196" alt="image" src="https://github.com/user-attachments/assets/abb572e9-e2dd-40b5-9db3-2f8863f3d6d9" />

---

### 4. Enter the repository directory:

   ```bash
   cd Qiskit-QCNN
   ```

   <img width="612" height="36" alt="image" src="https://github.com/user-attachments/assets/bfdb4dac-985d-4818-8a88-c648c00996fe" />

---

### 5. Start Jupyter Notebook:

   ```bash
   jupyter-notebook
   ```

   <img width="612" height="192" alt="image" src="https://github.com/user-attachments/assets/9bde0c1a-d0c0-45e9-ba2d-34e120934d66" />

---

### 6. Open your browser and navigate to:

   ```
   http://localhost:8888
   ```
   
   <img width="998" height="318" alt="image" src="https://github.com/user-attachments/assets/3f05c35e-854e-4eb2-a538-460c3c512c5f" />


   You should now see the notebook file contained in this repository (the .ipynb file). Click on one this to open the notebook and run the code examples.
